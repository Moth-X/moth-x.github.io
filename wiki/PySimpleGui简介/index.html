PySimpleGui简介

谈到Python的图形页面(GUI)库，很多人都会想到PyQt5等库，不可否认的是，PyQt5等库有着较为丰富的功能，但试想以下场景：当你实现了一个小型程序的功能，想要为这个程序加上GUI时，却发现还要在窗口布局、信号和槽等上面大费周章，耐心恐怕在打开QTdesigner时便已耗尽，此时你愤然点击右上角的“X”，说到：

“命令行又不是不能用！”

PySimpleGui可能就是为解决此类问题而生，它能满足你对快速构建GUI的小小要求，尽管他可能没有那么精美（花里胡哨），但没关系，我们只需要把我们的程序写成屎山，PySimpleGui自会变得精美，毕竟没有比较就没有伤害。

啊哦，无意冒犯，我的意思是：当我们享受PySimpleGui的简洁高效时，谁还会在意它能不能构建精美的界面呢？下面请看PySimpleGui的正经简介：

PySimpleGui 是一个用于 Python 的开源跨平台 GUI 库。 它旨在提供一个统一的 API，用于创建基于 Python 的 Tkinter、PySide 和 WxPython 工具包的桌面 GUI。 PySimpleGUI 也有一个用于 Remi 的端口，这对于为 Web 构建 GUI 很有用。 PySimpleGui 使您可以比直接使用它使用的库更快地构建 GUI。
